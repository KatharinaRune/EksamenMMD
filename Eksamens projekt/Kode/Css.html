CSS

CSS står for Cascading Style Sheets

Css bruges til at beskrive, hvordan man gerne vil have sit HTML-dokument vist f.eks i en browser.

Når man typografibeskrivelsen i et dokument for sig, kan men benytte det i flere dokumenter.

Laver man ikke en CSS og har sin typografibeskrivelse for sig selv, skal man gentage det samme stykke kode på flere sider, for at det har samme effekt.

Ved brug af CSS i et seberart dokument, kan man lade CSS styre alt udseende på hjemmesiden, imens HTML styre alt indholdet på siden.

CSS historie

Første version af CSS er udviklet af Håkon Wium Lie, senere blev CSS “adopteret” af W3C som herefter overtog standardiseringsarbejdet. Den første browser som understøtter CSS var IE3. Kort efter understøttede Netscape 4.0 også CSS, men havde mange fejl. Nu understøtter alle store browsere standarden.
