HTML og CSS workshop

Hvad er det?

HTML og CSS workshop, det var et projekt hvor vi i små grupper skulle fokusere på nogle forskellige emner, som vi fik tildelt. opgaven gik ud på at vi i små grupper skulle lave en hjemmeside som informerede om hvordan de forskellige emner kunne benyttes. Min makker og jeg fik emnet billeder som baggrund, og udarbejdede en hjemmeside som informerede om hvordan det kunne benyttes.

Hvad kom vi frem til?

Gruppevis skulle vi lave en hjemmeside, som illustrerede og forklarede brugen af det emne vi fik tildelt. Min makker og jeg fik tildelt brugen af billeder som baggrund og gradient. Vi fik lavet en side ud fra det.
